# 日记风格预览功能设计文档

> **日期：** 2026-06-04
> **状态：** 已批准
> **设计师：** Claude Code

## 概述

为设置页面的日记风格选择功能添加精美的预览体验，让用户在选择风格前能够直观地看到每种风格的实际呈现效果。

## 创意概念：维度观测站

用户在设置页面选择日记风格时，就像在「维度观测站」中预览不同平行宇宙的风景。每种风格都是一个独特的维度入口，点击预览按钮即可「穿越」到那个维度，体验真实的日记呈现效果。

**核心理念：**
- 沉浸式体验 - 让用户感受到每种风格的独特魅力
- 维度穿越 - 预览过程像是一次跨维度旅行
- 视觉叙事 - 通过视觉设计讲述平行宇宙的故事
- 微交互 - 精致的动画反馈增强用户体验

## 功能设计

### 1. 风格预览卡片

在现有的三种风格选择卡片（报纸汇编风、散文叙事风、荒诞编年史风）基础上，添加预览功能。

**卡片设计：**
- 保持现有的RadioCard样式
- 在卡片右上角添加「预览」标签（悬停时显示）
- 卡片悬停时上浮8px，边框发光，阴影扩散
- **点击行为：** 点击卡片任意位置打开预览模态框，选择风格需在模态框内点击「选择此风格」按钮

**悬停动画：**
```javascript
gsap.to(card, {
  y: -8,
  boxShadow: '0 20px 60px -15px rgba(167,139,250,0.3)',
  duration: 0.4,
  ease: 'power3.out'
});
```

## 视觉设计规范

### 颜色系统

沿用项目现有的暗色主题设计系统：

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 背景色 | `#0A0A0F` | void - 主背景 |
| 量子紫 | `#A78BFA` | 报纸汇编风主色 |
| 等离子青 | `#22D3EE` | 荒诞编年史风主色 |
| 星云粉 | `#F472B6` | 散文叙事风主色 |
| 信号白 | `#E5E5E5` | 主要文字 |
| 静态灰 | `#888888` | 次要文字 |

### 字体系统

| 用途 | 字体 | 说明 |
|------|------|------|
| 标题 | Playfair Display | 衬线字体，增加历史感 |
| 正文 | Inter | 无衬线字体，提高可读性 |
| 标签 | JetBrains Mono | 等宽字体，增加科技感 |

### 动画规范

**基础动画：**
- 持续时间：0.3-0.4秒
- 缓动函数：`cubic-bezier(0.4, 0, 0.2, 1)` 或 `power3.out`
- 动画属性：`transform` 和 `opacity`（保证60fps）

**尊重用户偏好：**
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 动画实现

使用GSAP实现所有动画效果：

```typescript
import gsap from 'gsap';

// 卡片悬停动画
const handleCardHover = (card: HTMLElement, isEntering: boolean) => {
  gsap.to(card, {
    y: isEntering ? -8 : 0,
    boxShadow: isEntering
      ? '0 20px 60px -15px rgba(167,139,250,0.3)'
      : 'none',
    duration: 0.4,
    ease: 'power3.out'
  });
};

// 模态框动画
const openModal = (modal: HTMLElement) => {
  const tl = gsap.timeline();
  tl.from(modal, { y: 100, opacity: 0, duration: 0.5 })
    .from('.modal-content', { y: 30, opacity: 0, stagger: 0.1 }, '-=0.3');
};
```

### 无障碍设计

- 所有按钮使用语义化HTML `<button>`
- 模态框支持键盘导航（Tab、Escape）
- 颜色对比度符合WCAG AA标准（4.5:1）
- 焦点状态清晰可见
- 屏幕阅读器友好的ARIA标签

## 交互流程

### 用户操作流程

```
1. 用户进入设置页面
   ↓
2. 看到三种风格选择卡片
   ↓
3. 悬停卡片，看到预览标签出现
   ↓
4. 点击卡片任意位置（或悬停后点击预览标签）
   ↓
5. 弹出模态框，显示该风格的完整预览
   ↓
6. 用户查看预览内容
   ↓
7. 点击「选择此风格」或「关闭预览」
   ↓
8. 如果选择，更新设置并关闭模态框；如果关闭，仅关闭模态框返回设置页面
```

### 状态转换

```
IDLE → HOVERING → PREVIEW_OPEN → SELECTED/CLOSED
  ↑         ↓            ↓              ↓
  └─────────┴────────────┴──────────────┘
```
